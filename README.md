# Site Romântico - Instruções

## 🎵 Áudio
Os arquivos de música estão em `musicas_site/`.
O player toca sequencialmente: `musica1.mp3`, `musica2.mp3`, etc.

**Para adicionar novas músicas:**
1. Coloque o arquivo original em `musicas_site/orig/`.
2. Rode o script de conversão: `python convert_audio.py`
   OU converta manualmente para .mp3 e renomeie para `musica3.mp3`, etc.
